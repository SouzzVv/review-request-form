# Review Request Form

A web-based tool designed to streamline internal review requests. It allows employees to standardize critical analysis data (P.S.Q, I.T.) and automatically export the records to a CSV file (Excel compatible).

## Features
- **Standardized Data Entry:** Custom fields for technical review codes.
- **Client-Side Export:** Generates a formatted `.csv` file directly from the browser using JavaScript.
- **User-Friendly Interface:** Designed in Portuguese (pt-BR) for local team usability.

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use
1. Clone this repository or download the files.
2. Open `index.html` in your browser.
3. Fill in the required fields (Request ID, Description, P.S.Q, etc.).
